ChiffonUpdater customization guide

The items below are easily customizable within the toolkit:
  • Display name of the agent
  • Display name of the update package
  • Icon of the agent
  • Icon of the update package
  • Banner of the update package
The icons and the banner must be in PNG format. In order to change these settings, you must use the Customization information section in your project.
For the files, the path must be either absolute or relative to the source folder (not possible if no source folder is used).
