Welcome to the ChiffonUpdater Project!

This project allows software editors to easily package their updates and distribute them to the end-users.

Key features of this project:
  • A graphical toolkit allowing software editors to easily register their products
  • High flexibility for product updates (support for proprietary modules enabling custom update tasks)
  • Support for backing up automatically version and rolling back products to a previous version
  • A pretty graphical interface to give to the end-users the best updating experience
  • Support for automatic generation of an update catalog allowing infrastructure administrators to update their products offline
  • Components can be customized (setting up a different icon or a different title)
